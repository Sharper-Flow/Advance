# ADV Runtime Protocol Coverage

> Source-of-truth coverage matrix for the lean runtime ADV agent. It proves runtime-protocol preservation without appending `ADV_INSTRUCTIONS.md` wholesale into the agent prompt.

## Coverage Rules

- Runtime-critical, agent-owned behavior stays in `.opencode/agents/adv.md` unless a code/spec/command contract fully owns it.
- Removed or compressed reference text must map to retained runtime text, code/spec enforcement, command-contract ownership, or reference-only material.
- Prompt-size metrics are reporting-only; they are not correctness proof.
