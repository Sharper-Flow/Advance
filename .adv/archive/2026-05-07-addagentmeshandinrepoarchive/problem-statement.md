ADV's durable intent and cross-project coordination both bypass git and GitHub Issues — the natural coordination surfaces. Two concrete symptoms: (1) `git blame` on archived code follows commit trailers (`Change: <id>`) into a dead link because the proposal lives only in `~/.local/share/opencode/plugins/advance/{project-id}/archive/`, never in the repo; (2) `adv_change_create target_path: <other-repo>` writes a change directly into another project's external state with no handshake, no consent, and no durable record outside that other project's local state. Both stem from the same root: ADV's external-state authority has no path back into git for durable artifacts and no path through GitHub Issues for cross-project coordination.
