# Improve Acceptance Reviews

## Intent
Improve ADV acceptance-review output formatting so dense remediation summaries are easier to scan. Add an executive summary and use ordered/nested lists where order or grouping matters.

## LBP Targets
- Identify where acceptance-review output is generated.
- Prefer shared voice/template guidance over one-off formatting hacks.
- Preserve existing gate handoff voice standards and command boundaries.

## Scope
- Acceptance review / accept workflow output templates.
- Command voice standard documentation if it governs the output.
- Adjacent tests/assets that enforce command copy or review output.

## Success Criteria
- Future acceptance review reports include a concise executive summary.
- Review details use structured verdicts and correctly nested ordered lists where order/grouping matters.
- Existing gate handoff voice standards remain intact.
- Formatting behavior is covered by nearby tests or asset checks where applicable.
