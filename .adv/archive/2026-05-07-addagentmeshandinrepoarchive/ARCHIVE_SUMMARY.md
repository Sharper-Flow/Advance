## Wisdom Accumulated

- **[gotcha]** Previous red evidence recorded a timeout because the test command used pipes (`pnpm test -- ... 2>&1 | tail -40`). When piping, the shell reports the exit code of the last command in the pipe (tail), not vitest. Use `npx vitest run` directly instead of `pnpm test` in worktrees, and avoid piping for test commands that need exit code accuracy.
